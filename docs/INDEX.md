## üìù Important Notes

- **All SQL migrations** should be run **individually** in Supabase SQL Editor
- For 3D models, we recommend **GLTF with external textures** (works always)
- **GLB embedded textures** require **development build** (don't work in Expo Go)
- Always check `.gitignore` before committing sensitive files
- Environment variables should be in `.env` file (not committed to git)
